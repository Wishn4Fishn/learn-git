Linux Tips, Hints, Usage

  • Network World: 2-Minute Linux Tips
    • chage
      Description: change age.
      Usage:
      chage -l yourusername
      - view your account settings
      sudo chage -l username
      - view another users settingd
      sudo chage -M number username
      - change the max life for a password
      sudo chage -m number username
      - change the minimum time before a password
      sudo chage -E yyyy-mm-dd username
      - set the expiration date
    • ln
      Description: Used to create a link to a file or driectory.
      Usage:
      ln -s filepath/filename
      - create a symbolic link to filepath/filename
      ln filepath/filename filename
      - create a hard link to filepath/filename. same file will exist in two places.
    • mv
      Description: rename a file
      Usage:
      move a file to a new location
    • pwd
      Description: present working directory
      Usage:
      pwd
      pwd -P
      - symlinks can make results misleading.
    • script
      Description: used to record commands and output
      Usage:
      script test1
      exit to end the recording
      recording and results in text file test1.
    • tac
      Description:same as cat but in reverse.
      Usage:
      tac filename
    • xz
      Description: file compression command
      Usage:
      xz filename
      - result is filename.xz
      unxz filename.xz
      - to uncompress
      xzcat filename.xz
      - to view the contents of a compress file.
      man pages for xz suggests using xz -d to decompress instead of unxz.
    Skipping tips on at and convert.
    - at has been deprecated and replaced with schtasks.
    - convert is from a imaging app and not the os command.
    - or operator, used in scripts
    - tab complete nothing new there
    - umask, not sure when i would use this.
  • 12 Linux Commands to Have Some Fun in the Terminal
    • Nothing really useful here but could be fun. These won't work in gitbash. And there are more than 12.
    1. Run a train in the terminal
      sudo apt install sl
      sl
      sl -F
    2. Matrix Terminal
      sudo apt install cmatrix
      cmatrix
      - crtl-c to stop it.
    3. Let the terminal speak
      sudo apt install espeak
      espeak "Type what your computer says"
    4. Calculate Factor
      factor whole number
    5. Let there be fire
      Now you are going to start a fire in your terminal!
      sudo apt install libaa-bin
      aafire
      - crtl-c to stop it.
    6. Fortune Cookie Command
      The terminal will display a random sentence just like you usually get in fortune cookies.
      sudo apt install fortune
      fortune
    7. Pet lover? This is for you
      Oneko is a little fun command that will change your regular cursor into a mouse and creates a curious little cat who will chase your cursor once you move it. It’s not limited to just the terminal. You can keep on working while the cat chases the cursor.
      sudo apt install oneko
      oneko
      oneko -dog
      - -dog to change it to a dog
      - crtl-c to stop it.
    8. Little brother is watching you
      Xeyes is a tiny gui program that let the user draws a pair of ever watching eyes! It will follow your mouse cursor constantly. Run the command and see it yourself!
      sudo apt install xeyes
      xeyes
      - crtl-c to stop it.
    9. Reverse a string
      rev
    10. Toilet (but it has nothing to do with a washroom)
      This sounds weird, yes. But, it’s just a command that transforms a text into large ASCII characters.
      sudo apt install toilet
      toilet sample text you want
    11. What does the fox cow say?
      Cowsay is a command that displays a cow using ASCII characters in the terminal. And by using this command you can instruct the cow to say anything you want.
      sudo apt install cowsay
      cowsay "your text"
    12. Banner Command
      The banner command works just like the toilet command but it is limited to print only 10 characters at most.
      sudo apt install sysvbanner
      banner "Welcome"
    13. Yes command
      Infinite loop of "your text".
      yes "your text"
      - crtl-c to stop it.
    14. Get a new identity, well, sort of
      Want to generate a random fake identity? I give you the command “rig”. Once you place this in the terminal, it will generate a fake identity.
      sudo apt install rig
      rig
    15. Command: bb
      See what happens?
      apt-get install bb
      bb
